# 💰 AI Finance Advisor

An AI-powered personal finance advisor built with **Python**, **Streamlit**, and **APIs**. This application helps users manage their finances through an interactive dashboard, AI-powered financial guidance, budgeting tools, expense tracking, investment insights, and visual analytics.

---

## 🚀 Features

- 🤖 AI Financial Advisor (Google Gemini AI)
- 🔐 User Authentication (Login & Signup)
- 📊 Interactive Dashboard
- 💸 Expense Tracking
- 💰 Budget Planning
- 📈 Investment Suggestions
- 📉 Financial Reports & Analytics
- 📅 Monthly Expense Summary
- 📂 Data Management
- 🌙 Modern User Interface
- ⚡ Fast and Responsive Streamlit App

---



### Libraries Used
- Streamlit
- Pandas
- Plotly
- Matplotlib
- Google Generative AI
- python-dotenv
- SQLite3
